# Voxi

Lightweight Windows TTS tray app using SAPI.

## Features

- Read clipboard text aloud via hotkey
- Cycle through system voices
- Adjustable speech speed
- Built-in pronunciation dictionary for URLs, acronyms, emojis

## Hotkeys

- `Ctrl+Shift+1` - Toggle read/stop
- `Ctrl+Shift+2` - Cycle speed
- `Ctrl+Shift+3` - Cycle voice
- `Ctrl+Shift+Q` - Exit

## Build

```
cargo build --release
```
